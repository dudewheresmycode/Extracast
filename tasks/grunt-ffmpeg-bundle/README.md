##grunt-ffmpeg-bundle
---
Bundle platform specific ffmpeg binaries into your project

###Options

**platform**
Which static build to bundle. Accepted values are:

  auto (default)
  osx-64
  linux-32
  linux-64
  linux-armel
  linux-armhf
  windows-32
  windows-64


Thanks to [ffbinaries.com](http://ffbinaries.com) for providing the API!
